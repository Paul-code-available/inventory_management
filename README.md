# Inventory Management

Sistema de gestión de inventario desarrollado con Spring Boot.

## Tecnologías

- Java 21
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Spring Security

## Requisitos

- JDK 21
- Maven
- MySQL 8+

## Cómo ejecutar

1. Clonar el repositorio.
2. Crear la base de datos:

```sql
CREATE DATABASE inventory_management;
```

3. Configurar `application.properties`.

4. Ejecutar:

```bash
./mvnw spring-boot:run
```
